# Demo Script

## English Demo Call

1. Open `http://localhost:3000/simulator`.
2. Select `Interested` and `English`.
3. Run the simulation.
4. Show the EMI branch, document reminder, hot-lead escalation, transcript, and
   latency breakdown.

Suggested customer line:

> I am interested, but can you explain the EMI and documents needed?

## Hindi Demo Call

1. Select `Hindi callback` and `Hindi`.
2. Run the simulation.
3. Show Hindi/Hinglish response, callback booking, lead status, and intent label.

Suggested customer line:

> Namaste, meri EMI kitni hogi aur kal 4 pm callback kara dijiye.

## Recorded Video Checklist

- Minimum 3 minutes
- Show dashboard metrics
- Run one English simulation
- Run one Hindi simulation
- Open Settings and show model tier/provider status
- Open Calls and show transcript/latency
